Real-time backup status

Keep an eye on the status of each of your backups.
Check their size, dates, and several other stats.

Easy configuration

Add as many backup sets as you need, with different target paths and destinations.

File tree

View all the files contained in a backup, for an easy monitoring and quick restore operations.

Scheduler

Configure and combine schedules accurately, for always up-to-date backups.

Vuplicity is powered by web technologies, and runs on Atom Electron.

The source code is hosted on Github.
